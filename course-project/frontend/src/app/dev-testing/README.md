# Development Testing Playground

This directory contains a testing page for development purposes only.

## Access

In development mode, visit: `http://localhost:3000/dev-testing`

## Purpose

This page is a playground for testing and demonstrating UI components during development. Add your component tests here!

Currently includes:
- **Symbol Component**: Various symbols with different sizes and colors
- **Primary Button**: All variants including text-only, with symbols, custom colors, text colors, outlined, and disabled states
- **Secondary Button**: All variants including text-only, with symbols, custom colors, text colors, and disabled states
- **Real World Examples**: Common UI patterns using combinations of buttons

## Production

This page is automatically excluded from production builds and will redirect to 404 in production environments.

## Usage

1. Add new component tests to this page as you develop them
2. Click any button to see console logs and alerts demonstrating the onClick handlers
3. Use this page to verify component behavior before integrating into the main app
